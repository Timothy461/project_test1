# КОДИТОРГ — лендинг

Лендинг компании по сопровождению предпринимателей в госзакупках (44-ФЗ, 223-ФЗ). Next.js 14 + TypeScript + Tailwind CSS.

## Запуск локально

```bash
npm install
npm run dev
```

Откроется на `http://localhost:3000`.

## Структура

```
app/            — страница и layout (App Router)
components/     — lead-modal.tsx и error-boundary.tsx (см. примечание ниже)
lib/data.ts     — контент: услуги, статистика, отзывы
```

## Что уже исправлено по сравнению с исходником

- Ссылки на изображения услуг в `lib/data.ts` использовали `source.unsplash.com` — этот сервис Unsplash отключён с 2021 года. Заменены на рабочие прямые ссылки `images.unsplash.com`.

## Что не исправлено — на ваше решение

- `trustPoints` и `testimonials` в `lib/data.ts` импортируются в `app/page.tsx`, но не рендерятся — секции «Доверие» и отзывов нет на странице, хотя контент для них готов.
- Пункт меню «Доверие» ссылается на `#trust` — такого `id` на странице нет.
- `components/lead-modal.tsx` (с валидацией полей) и `components/error-boundary.tsx` (настоящий Error Boundary) не используются: `app/page.tsx` определяет свои упрощённые версии модалки и error boundary прямо внутри файла.

Подробнее о цветах, типографике, отступах и мобильных требованиях — в `docs/design.md`. Статичное HTML-превью (как видно без сборки проекта) — в `preview/koditorg-landing-preview.html`.

## Деплой

Подойдёт любой хостинг, поддерживающий Next.js (Vercel, Netlify и т.п.). Перед продом стоит решить открытые вопросы выше и заменить фото услуг на брендовые.
